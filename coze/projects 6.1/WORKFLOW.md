# 塑料价格监控系统 - 需求工作流

## 一、项目背景

**用户画像**：学生团队，开发塑料原料价格监控系统

**核心目标**：监控普拉司网站（plasway.com）的塑料原料价格变动，及时提醒用户关注的价格变化

---

## 二、功能需求

### 2.1 价格监控

| 需求项 | 描述 | 优先级 |
|--------|------|--------|
| 数据来源 | 普拉司网站（plasway.com）店铺商品价格 | P0 |
| 更新频率 | 15分钟自动刷新 + 手动刷新按钮 | P0 |
| 价格取值策略 | 用户可选择：最低价、第N低价、最高价等 | P1 |
| 商品筛选 | 按品名/制造商筛选商品 | P1 |
| 型号显示 | 显示商品型号字段 | P0 |

### 2.2 两级提醒系统

| 提醒级别 | 触发条件 | 提醒方式 | 音效 |
|----------|----------|----------|------|
| **一般提醒** | 价格变动 ≥ 一般阈值（默认2%） | 应用内蓝色标记显示 | 无 |
| **紧急提醒** | 价格变动 ≥ 紧急阈值（默认5%） | 红色标记 + 弹窗 + 音效 | 有（类似微信） |

### 2.3 提醒阈值设置

| 设置项 | 说明 |
|--------|------|
| 一般阈值 | 默认2%，可自定义 |
| 紧急阈值 | 默认5%，可自定义 |
| 阈值类型 | 百分比 或 绝对金额 |
| 静默时段 | 可设置免打扰时间段（如22:00-08:00） |

### 2.4 店铺管理

| 功能 | 描述 |
|------|------|
| 添加店铺 | 输入普拉司店铺URL，一键自动爬取商品 |
| 店铺列表 | 查看已添加的所有店铺 |
| 刷新数据 | 一键刷新店铺商品价格 |

### 2.5 品类管理

| 功能 | 描述 |
|------|------|
| 商品列表 | Tab按品类（ABS/PP/PE等）分组显示 |
| 监控开关 | 单个商品开启/关闭监控 |
| 批量操作 | 批量开启/关闭监控 |
| 自定义阈值 | 单个商品可设置独立阈值 |

### 5.2 提醒判断逻辑

```typescript
function getAlertType(changePercent: number, rules: AlertRules): AlertType {
  const absChange = Math.abs(changePercent);
  
  // 检查是否在静默时段
  if (isInQuietHours(rules)) {
    return 'silent'; // 仅记录，不提醒
  }
  
  // 判断提醒级别
  if (absChange >= rules.urgentThreshold) {
    return 'urgent'; // 紧急提醒
  }
  
  if (absChange >= rules.generalThreshold) {
    return 'general'; // 一般提醒
  }
  
  return 'none'; // 无需提醒
}
```

## 七、已知限制与解决方案

| 限制 | 说明 | 解决方案 |
|------|------|----------|
| 爬虫IP封锁 | 普拉司网站可能封锁服务器IP | 1. 使用代理IP池 2. 降低请求频率 3. 联系网站申请API权限 |
| 数据延迟 | 15分钟更新一次，非实时 | 可接受，塑料价格变动不频繁 |
| 网络依赖 | 需要稳定的网络连接 | 添加重试机制，失败自动重试3次 |
